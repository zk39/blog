# SignUp Event Pages

## Introduction

My project is a web application that allows users to browse and register for events in different locations. The main page displays a map with pins indicating the locations, each with an associated picture. When a user clicks on a picture, they are taken to a page that displays a list of events for that location. Clicking on an event takes the user to a page with details about the event and the option to register. The project is built using Next.js, with routing and local database to manage the data and API calls.

Here is the link [SignUp Events](https://next-js-sign-up-events-owbn2w8fw-zk39.vercel.app/)

## Technologies Used

 - Next.js 12
 - Static Site Generation
 - Routing
 - Local database
 - API
 - SASS
 - Flexbox

Next.js was used to build the project, with routing and API calls to manage the data. A local database was used to store user registrations for events. Static Site Generation improved website performance, SASS was used to style the pages, with flexbox to arrange the layout.

## Conclusion
This project is a web application that allows users to browse and register for events in different locations. The use of Next.js, routing, local database, Static Site Generation and API calls made it possible to create a dynamic and responsive application. The styling with SASS and layout with flexbox provides an easy-to-use interface for users. With further development, additional features can be added to enhance the functionality of the application.
